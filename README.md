# AutoCheck Car Diagnosis 🚗🔧

AutoCheck Car Diagnosis is a modern, investor‑ready landing page designed to showcase vehicle diagnostic services with professional branding, smooth navigation, and interactive features.

---

## 📌 Features
- **Sticky Navbar** with smooth scrolling for easy navigation
- **Hero Section** with animated slogan and call‑to‑action
- **Services Section** highlighting diagnostic offerings
- **Pricing Section** with clear visual hierarchy (Basic, Standard, Premium)
- **Testimonials & Why Choose Us** for credibility and trust
- **About Us** mission statement
- **Contact Section** with styled form, date/time picker, and embedded map
- **FAQs Section** with accordion interaction
- **Footer** with quick links and glowing social icons
- **Back to Top Button** for improved user experience
- **Micro‑interactions** (hover effects on buttons/icons)
- **Angled & Wave Dividers** for modern section transitions

---

## 📌 Tech Stack
- **HTML5**  
- **CSS3 (Bootstrap + custom styles)**  
- **JavaScript (for interactivity)**  
- **Font Awesome** (icons)  
- **GitHub Pages** (deployment)
